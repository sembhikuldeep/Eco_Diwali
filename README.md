# 🌿 EcoDiwali Waste Advisor

EcoDiwali Waste Advisor is a simple web application that helps users manage festival waste responsibly by suggesting reuse and recycling ideas.

The project promotes **eco-friendly celebrations** by encouraging people to reuse common Diwali waste items instead of throwing them away.

---

## 🚀 Features

- Provides reuse and recycling suggestions for common Diwali waste
- Simple and clean user interface
- Built using Python and Flask
- Interactive web page
- Encourages eco-friendly festival practices

---

## 🛠 Technologies Used

- Python
- Flask
- HTML
- CSS
- Jinja2 Templates
